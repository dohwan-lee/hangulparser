# hangulparser
Find Korean and create an Excel document.

혼자서 다국어 처리를 하게 되었다. 물론 도움을 요청해야 겠지만 프로그램을 통해서 다국어 처리를 할 수 있다면 비용을 아낄 수 있다.
스프링의 i18n을 이용한 프로퍼티 파일 활용과 {spring:message}로 적용하도록 하고 싶다.
